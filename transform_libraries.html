The Data Transformation Libraries enable developers and data scientists to deliver integration and analytics.

 

Portable

The libraries utilize the Microsoft .Net core platform and can be used to build applications for Windows, Mac, Linux, IOS and Android.

Logical and Sequential

Gone are spaghetti code and complex integrations. The information hub provides a best-practice data management framework out of the box, that enables immediate loading, cleansing, and transformation of data.

Fast & Scalable

The simplicity and the built in framework, the scalability of the cloud, and the capacity capabilities of the "no sql" databases allows this platform to manage 100's of terabytes of data.

Built in capabilities

The hub provides built in capabilities including an automatic staging builder, 120+ built in functions, a job scheduler, and reusable data validation rules.

Secure

The Hub recognises data is sensitive. Source data fields can easily be tagged as sensitive, and this data will be safely encrypted before it is stored in the cloud.

Power Users

Some data and business problems can be complex. In order to facilitate this, developers integrate their own c# code, and leverage the engine and interface to solve almost any data problem.
