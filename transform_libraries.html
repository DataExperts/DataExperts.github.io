The Data Transformation Libraries enable developers and data scientists to deliver integration and analytics.

 

Portable

The libraries utilize the Microsoft .Net core platform and can be used to build applications for Windows, Mac, Linux, IOS and Android.

Full suite of transforms

Group, Sort, Pivot and Join data sets from heterogeneous data sources on the fly. Our extensive set of transforms enables virtually any type of data transformation processing.

Built in functions

There are hundreds of built in functions to map, aggregate, and parse your data, or write your own c# functions to do virtually anything.

Runs Fast

By using compiled code, optimized built in functions and native database drivers, you can easily process hundreds of thousands of rows per second.

Secure

Built in security features make field level encryption and hashing as simple as flagging a field.

Advanced Tools

Advanced transforms enable change data capture and history preservation.
